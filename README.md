# Install
- ```git clone```
- ```git submodules init```
- ```git submodules update```
- then stow the appropriate directories

# tmux configuration
- ```stow tmux```
- launch tmux
- ```prefix + I``` => will install the appropriate plugins (copycat, etc.)

# alacritty configuration
```stow alacritty```

# other
- works best with ```UbuntuMono Nerd Font``` installed
